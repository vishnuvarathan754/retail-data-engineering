# 🛍️ Retail Data Engineering Project

## 📌 Overview
This project analyzes retail store data to improve sales, inventory, and customer insights using Databricks and PySpark.

## 🚀 Tools Used
- Python
- PySpark
- Databricks

## 📊 Features
- Sales analysis (top-selling products)
- Inventory optimization (low stock, overstock)
- Customer behavior analysis

## 💡 Business Impact
- Improves product availability
- Supports better stock decisions
- Enhances customer experience

## 🧠 Key Learning
- Fixed file format issues (CSV vs Excel)
- Handled column naming problems
- Solved data type conversion issues
